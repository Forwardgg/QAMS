=== AUTH CONTROLLER ===
    registerUser() - Registers new user with email validation and role checking
    loginUser() - Authenticates user and returns JWT token
    forgotPassword() - Initiates password reset, sends reset link (logs to console)
    resetPassword() - Validates reset token and updates password
    deleteUser() - Soft deletes user by setting status to inactive
    forceDeleteUser() - Hard deletes user from database

=== COURSE CONTROLLER ===
    getAllCourses() - Gets paginated courses with search and sorting
    getCourseById() - Gets single course by ID
    getCourseByCode() - Gets single course by code
    createCourse() - Creates new course with validation
    updateCourse() - Updates course fields with conflict checking
    deleteCourse() - Deletes course and returns deleted record
    searchCourses() - Searches courses with query and pagination

=== COURSE OUTCOME CONTROLLER ===
    getAllCourseOutcomes() - Gets paginated course outcomes with ordering
    getCourseOutcomesByCourseCode() - Gets outcomes by course code
    getCourseOutcomeByNumber() - Gets specific outcome by course ID and CO number
    getCourseOutcomeById() - Gets outcome by its ID
    createCourseOutcome() - Creates new course outcome with validation
    updateCourseOutcome() - Updates outcome fields with conflict checking
    deleteCourseOutcome() - Deletes outcome and returns deleted record
    searchCourseOutcomes() - Searches outcomes by course code and CO number

=== QUESTION PAPER CONTROLLER ===
    createQuestionPaper() - Creates new question paper with validation
    getAllPapers() - Gets all papers with pagination
    getPapersByCourse() - Gets papers by course code
    getPapersByCourseAndCO() - Gets papers by course code and CO number
    updatePaper() - Updates paper fields with ownership validation
    submitForModeration() - Submits paper for moderation with ownership and status checks
    deletePaper() - Deletes paper with ownership validation

=== QUESTION CONTROLLER ===
    createQuestion() - Creates new question with paper access validation and media linking
    updateQuestion() - Updates question with transaction support and media reconciliation
    getQuestion() - Gets question by ID with media details
    getQuestionsByPaper() - Gets all questions for a paper in sequence order
    deleteQuestion() - Deletes question and handles cleanup
    searchQuestions() - Searches questions with filters and pagination
    updateQuestionSequence() - Updates question ordering within a paper

=== MODERATOR CONTROLLER ===
    getPapersForModeration() - Gets papers available for moderation with filters
    getPaperDetails() - Gets paper details with questions and moderation status
    startModeration() - Starts moderation process with transaction support
    updateQuestionStatus() - Updates individual question status
    bulkUpdateQuestionStatus() - Bulk updates question statuses with validation
    submitModerationReport() - Submits final moderation report with decision
    getModerationHistory() - Gets moderation history for current moderator
    getCOBreakdown() - Gets questions grouped by CO for moderation
    viewQuestionReport() - Gets questions with full content for reports
    viewPaperReport() - Gets comprehensive paper report with moderation data
    getAllModerations() - Gets all moderations with search/filters (admin only)
    getModerationDetails() - Gets detailed moderation record by ID
    generateModerationReportPdf() - Generates PDF moderation report

=== UPLOAD CONTROLLER ===
    uploadFile() - Handles file upload with validation, signature checking, and media record creation
    getUploadConfig() - Returns upload configuration (allowed types, max file size, upload URL)

=== PDF CONTROLLER ===
    generatePdf() - Generates PDF from HTML or paper data with access control and validation
